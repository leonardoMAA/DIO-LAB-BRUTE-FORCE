# DIO-LAB-BRUTE-FORCE
Este repositório documenta a criação de um laboratório prático para simulação de ataques de força bruta em um ambiente controlado, utilizando:

->Kali Linux (máquina atacante)
->Metasploitable 2 (máquina vulnerável)
->DVWA (Damn Vulnerable Web Application)
->Ferramenta Medusa

O objetivo é compreender técnicas de ataque, validar vulnerabilidades e propor medidas de mitigação.

Objetivos de Aprendizagem:

->Compreender ataques de força bruta (FTP, Web, SMB)
->Utilizar o Kali Linux para auditoria de segurança
->Aplicar a ferramenta Medusa em diferentes cenários
->Documentar processos técnicos
->Identificar vulnerabilidades e sugerir mitigação

Ambiente do Laboratório:

->VirtualBox
->ISO do Kali Linux
->Metasploitable 2
->DVWA configurado
